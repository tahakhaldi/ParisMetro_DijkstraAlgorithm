# ParisMetro_DijkstraAlgorithm
Finding the shortest path between stations in a subway system (in Paris) using Djikstra's Algorithm.

## How to? 
Compiling your java files in directory

```
javac *.java
```
Running sample with results

```
java ParisMetro 103
```
